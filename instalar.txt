O Pronto utiliza o Banco de Dados PostgreSQL.

Para instalar siga as seguintes instruções:

1. Instale o Java 6.0 ou mais recente
http://www.java.com (o comando java deverá estar disponível na linha de comando - path)

Para saber mais sobre o que é path leia http://vlaurie.com/computers2/Articles/environment.htm se você usa windows ou http://www.vivaolinux.com.br/artigo/O-que-e-PATH-como-funciona-e-como-trabalhar-com-ele se você usa linux.

2. Instale o Maven 3 ou mais recente.
http://maven.apache.org

O comando mvn deverá estar disponível no path também.

3. Faça download do projeto através do git hub:
git clone git://github.com/andrefaria/pronto-agile.git

(se você usar windows, pode fazer o download em zip na url: https://github.com/andrefaria/pronto-agile/zipball/master)

4. Faça o download e instale o PostgreSQL para o seu sistema operacional em:
>> http://www.postgresql.org/download/

5. Execute o postgres apartir da pasta etc (que contém os scripts) com o usuário e senha padrão:
>> ex: psql postgres postgres;
Obs: O nome do aplicativo pode ser psql, psql83, psql84, etc, de acordo com a instalação.

6. Execute os seguintes comandos para criação do banco de dados.
create role pronto with login password 'pronto';
create database pronto owner pronto;
\c pronto pronto
\i pronto.sql 
\i pronto-updates.sql 
\i pronto-demo.sql 
\q

É importante certificar-se de que todos os objetos criados no banco pertençam ao usuário pronto.

7. Pronto, é só executar:
mvn jetty:start
obs: da pasta raiz do projeto (a mesma que contém o arquivo pom.xml)

8. Acesse em http://localhost:8080/pronto-agile ou http://127.0.0.1:8080/pronto-agile

Utilizar os usuários:
---------------------------------------------------
|Papel			| Usuáio	| Senha   |
---------------------------------------------------
|Product Owner		| justus	| justus  |
|Scrum Master		| coach		| coach	  |
|Equipe 		| time		| time 	  |
|Cliente 		| cliente 	| cliente |
---------------------------------------------------

8. Para Parar
mvn jetty:stop
obs: da pasta raiz do projeto (a mesma que contém o arquivo pom.xml)
